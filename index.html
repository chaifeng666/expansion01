Google 試算表串接指南 & 設定

1 步驟 1：建立 Google 試算表與部署 Apps Script

請至您的 Google 雲端硬碟新建一個試算表，點擊選單的 「擴充功能」 -> 「Apps Script」，並將下列程式碼完整複製並覆蓋取代：

/* Google Apps Script 接收端 */
function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    
    // 檢查標題列，如果是空表則自動建立
    if (sheet.getLastRow() === 0) {
      sheet.appendRow([
        "時間戳記", "領款人", "出席費", "交通費", "合計金額", 
        "身分證字號", "連絡電話", "戶籍住址", "日期", "簽名(Base64)"
      ]);
    }
    
    // 寫入資料
    sheet.appendRow([
      new Date(),
      data.payeeName,
      data.attendanceFee,
      data.transportFee,
      data.totalFee,
      data.idNumber,
      data.phone,
      data.address,
      data.dateStr,
      data.signature
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({ status: "success" }))
      .setMimeType(ContentService.MimeType.JSON)
      .setHeader("Access-Control-Allow-Origin", "*");
      
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({ status: "error", message: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON)
      .setHeader("Access-Control-Allow-Origin", "*");
  }
}

2 步驟 2：部署為網頁應用程式

在 Apps Script 編輯器右上角點擊「部署」 -> 「新增部署」：

  • 選取類型：網頁應用程式 (Web App)
  • 專案說明：填寫任何名稱
  • 誰可以存取：選取「所有人」 (Anyone) (此選項非常重要，否則網頁無法傳遞資料)
  • 部署成功後，複製畫面上產生的「網頁應用程式網址 (Web App URL)」並貼在下方設定欄。

※ 填寫後將儲存於您的瀏覽器記憶體中，每次填寫領據即可直接提交。
